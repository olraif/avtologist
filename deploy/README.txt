Загрузка на Netlify

1. Откройте https://www.netlify.com/
2. Войдите в аккаунт.
3. Выберите Add new site -> Deploy manually.
4. Перетащите СОДЕРЖИМОЕ этой папки deploy в окно загрузки.

Файлы для загрузки:
- index.html
- styles.css
- app.js
- manifest.webmanifest
- sw.js

После загрузки Netlify даст ссылку вида https://name.netlify.app
