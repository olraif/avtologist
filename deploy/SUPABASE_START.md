# Запуск сервера для Автологиста

Это самый дешевый путь для MVP: сайт остается на Netlify, а Supabase дает базу, пользователей и хранение грузов.

## 1. Создать проект

1. Откройте `https://supabase.com`.
2. Создайте аккаунт или войдите.
3. Нажмите `New project`.
4. Название: `avtologist`.
5. Region можно выбрать ближайший доступный европейский.
6. Database password сохраните у себя. В приложение пароль вводить не нужно.
7. Дождитесь создания проекта.

## 2. Создать таблицы

1. В Supabase откройте `SQL Editor`.
2. Нажмите `New query`.
3. Откройте файл `supabase_schema.sql`.
4. Скопируйте весь SQL.
5. Вставьте в Supabase и нажмите `Run`.

После этого появятся таблицы:

- `profiles` — профиль перевозчика;
- `vehicles` — машины;
- `sources` — источники грузов;
- `loads` — заявки на грузы;
- `responses` — отклики;
- `trip_events` — события рейса.

## 3. Где взять ключи для сайта

1. Откройте `Project Settings`.
2. Откройте `API`.
3. Нужны два значения:
   - `Project URL`;
   - `anon public key`.

`service_role key` в сайт вставлять нельзя. Это секретный ключ только для сервера.

## 4. Что делать потом

Следующий технический шаг: подключить фронт к Supabase. Тогда грузы, машины, источники и отклики будут сохраняться не только в телефоне, а в базе.

Для демо можно сначала подключить только:

1. сохранение машины;
2. импорт грузов;
3. список грузов;
4. отклики.
